# Handoff Update Checklist

단계 완료 전 다음을 확인한다.

- [ ] 단계명과 목적
- [ ] 입력 기준 ZIP/기준선
- [ ] 수정 파일
- [ ] 수정하지 않은 보호 파일
- [ ] 정적 검증 결과
- [ ] 사용자 로컬 실행 결과
- [ ] 현재 증거 수준
- [ ] 알려진 실패와 보정 이력
- [ ] 남은 리스크/보류 사항
- [ ] remote 적용 여부
- [ ] 다음 권장 단계
- [ ] 정확한 재개 지점
- [ ] 결과 ZIP 파일명
- [ ] 결과 ZIP SHA-256

미확인 항목은 `미확인`, 실행하지 않은 항목은 `미실행`으로 기록한다.
